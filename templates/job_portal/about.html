Our Client Projects: Delivering Excellence Across Industries

Sevendyne has successfully completed a range of innovative projects for clients across various sectors. Below are some highlights:

  • Python OpenAI based Zoho Recruit Project: Led the development of an AI-driven Zoho Recruit app using Python and OpenAI. The app automates job search processes via OpenAI prompts and integrates results into a web view through Zoho Recruit APIs. Oversaw the team involved in API integration and backend development.
  • Spree Ecommerce based Ruby on Rails Project: Managed a custom ecommerce dashboard project using Spree and Ruby on Rails. This platform scrapes data from existing ecommerce sites, utilizes OpenAI for automation, and creates new ecommerce platforms based on the gathered information. Guided the team in developing scraping modules and implementing dashboard features.
  • Angular Restaurant based Analytic Dashboard Project: Directed the creation of a data analytics dashboard for restaurant chains using Angular and SpringBoot Java. The dashboard visualizes customer data through Apache Echarts. Supervised the team in implementing various analytic graphs and front-end components.
  • CodeIgniter based CRM Project: Oversaw the development of a custom CRM system for RedDotPayment, focusing on payment management. This included building EMI payment modules and a demo QR payment app using Android and CodeIgniter PHP. Led the technical and management teams throughout the project lifecycle.
  • Laravel Vue.js based Property Management Project: Managed the development of a property and shipment management system using Laravel and Vue.js. Directed the team in creating CRUD forms, integrating REST API calls, and managing the front-end using Vue.js and Poppler.
  • C++ Qt based CodeReview Project: Led the development of a custom code review tool for Linux, including features such as project folder parsing and streamlined code review functionalities. Supervised a small team responsible for feature implementation and UI development.
  • C++ Qt based TradingSystemAPI Project: Managed the development of a trading platform UI using Qt for TradingSystemAPI. Directed the team in implementing chart visualizations and managing trading data presentation.
  • C++ Wt based WiseOwl Property Management Project: Led the development of a property management dashboard, focusing on tenant and broker functionalities. Supervised the team in integrating property listings and administrative features.
